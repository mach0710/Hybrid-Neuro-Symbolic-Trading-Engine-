
##  Overview
**Trading Planner AI** is a hybrid intelligence engine that combines **Machine Learning** (for pattern probability) with **Multi-Agent Reasoning** (for strategic decision-making). It uses a graph of specialized AI agents (Technical, Fundamental, Risk) to analyze markets and generate actionable trade plans in real-time.



##  Key Features
* **Hybrid Logic:** Validates ML breakout scores with LLM-based pattern recognition.
* **Multi-Agent System:** Dedicated agents for Technicals, News/Fundamentals, and Risk Management.
* **FastAPI Backend:** Asynchronous, non-blocking API for low-latency inference.
* **Gemini 1.5 Flash:** Powered by Google's latest model for speed and cost-efficiency.
